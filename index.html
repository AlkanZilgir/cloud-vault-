How to set up (free, 2 min)

Your files are stored on Cloudinary — a free professional cloud service. Sign up once, use forever.

  1. Go to cloudinary.com → and sign up free
  2. In your dashboard, copy your Cloud name
  3. Go to Settings → Upload → Upload presets
  4. Click Add upload preset, set signing to Unsigned, save it
  5. Copy the preset name and paste both below
⚠️ About deleting files: removing a file from your vault removes it from your gallery view, but the file stays on Cloudinary's servers. To fully delete, go to your Cloudinary Media Library. This is a limitation of free unsigned access.
